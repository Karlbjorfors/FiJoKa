> **Portfolio Project** - This is a refined version of a group project originally developed during my studies at Lexicon. I've adapted and enhanced it for my personal portfolio.

[View Live Application](https://github.com/Karlbjorfors/FiJoKa)

## 📖 About

FIJOKA is a comprehensive recipe management application that helps users organize their cooking life through:

✅ **Recipe Database** - Extensive collection of recipes with detailed instructions

✅ **Weekly Meal Planner** - Plan your meals for the week ahead

✅ **Automatic Shopping Lists** - Generate shopping lists based on your meal plans

✅ **Nutrition Calculator** - Calculate nutritional content per serving

✅ **Search & Filter Functions** - Find recipes that match your preferences

✅ **User Accounts** - Save your favorite recipes and meal plans

## 🏗️ Project Features

### Core Functionality

1. **User-Friendly Recipe Platform** - Browse and search through detailed recipes with step-by-step instructions
2. **Weekly Meal Planning** - Plan your meals for easier household management
3. **Automated Shopping Lists** - Generate shopping lists based on selected recipes
4. **Personal Recipe Collection** - Add your own recipes to build a personal database
5. **Nutrition Calculations** - Calculate nutritional content per serving including calories, protein, carbs, and fat

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Firebase account (for backend services)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Karlbjorfors/FiJoKa.git
   cd FiJoKa
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file in the root directory and add your Firebase configuration:

   ```bash
   NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser
